# ajiang-ppt-skill

> 横向翻页网页 PPT 生成 Skill

## 简介

这是一个用于生成高质量横向翻页网页 PPT 的 Skill，基于现代前端技术栈构建交互式演示网站。

## 技术栈

- **React 19** + Vite
- **Tailwind CSS v4**
- **Framer Motion**（动画）
- **Lucide React**（图标）

## 风格选项

| 风格 | 特点 |
|------|------|
| 电子杂志风 | 衬线字体 + 流体背景 + 暖色调 |
| 瑞士国际主义风 | 无衬线字体 + 网格点阵 + 高对比色 |

## 使用方法

在 SOLO 中调用此 Skill，描述你的 PPT 需求：

```
创建一个关于[主题]的PPT，[风格]，[页数]页
```
